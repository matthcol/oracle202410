#### Oracle tools
- sqldeveloper: GUI admin et requetage SQL
- sqlplus: CLI admin et requetage SQL
- rman: CLI backup et restauration
- lsnrctl: CLI gestion listener
- setup, dbca, netca, netmanager: outils d'installation ou configuration
